# EXOS Claims AI

Custom Frappe/ERPNext app for EXOS insurance claims validation with AI-assisted document review, approvals, and audit workflows.

## Install (Frappe Cloud / bench)

```bash
bench get-app https://github.com/ibra-a/exos_claims_ai.git
bench --site <site> install-app exos_claims_ai
bench --site <site> migrate
```
